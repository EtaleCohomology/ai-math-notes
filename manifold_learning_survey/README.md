# Manifold Learning Survey

ビッグデータに対する可微分多様体変換手法の体系的な調査報告書。

## 対象読者

- 機械学習・データサイエンスの研究者・実務家
- 産業応用における多様体手法の選択に関心のある方

## 内容

高次元・非線形なデータ構造を捉えるための可微分多様体アプローチについて、基礎理論から最新アルゴリズムの比較評価、産業応用の事例分析までを包括的に整理した。

主要トピック:

- 可微分多様体の基礎理論(接空間、リーマン計量、データ解析で用いられる主要な多様体)
- ビッグデータへの適用手法(多様体学習、リーマン最適化、測地線補間)
- 主要アルゴリズムの比較(UMAP、Riemannian SGD、Poincaré Embedding、SPDNet、GeoFlow)
- ベンチマーク評価(MNIST、WordNet、単細胞RNA-seq)
- 応用事例(自然言語処理、コンピュータビジョン、生命科学、金融工学)
