# Emergent Communication

AIエージェントが協力タスクの中で自発的にコミュニケーション・プロトコルを「発明」する現象に関する研究分野(Emergent Communication)に関するサーベイ論考です。

2016年のFoerster et al. を起点とする本研究分野の成立経緯、主要論文の時系列推移、キーパーソン、研究者自身が語る実用性、そして既存論文では言及されていない実用可能性について整理しました。

## 文書

📄 [emergent_communication_JA.pdf](./emergent_communication_JA.pdf)

## 収録内容

- 研究領域名と位置づけ
- キーパーソン(Lazaridou, Baroni, Foerster, Kottur, Mordatch, Batra 他)
- 主要論文の時系列(2016-2024)
- 論文執筆者たちが語る実用性
- 論文執筆者が言及していない実用可能性(6項目)
- 付録: LangGround (NeurIPS 2024) の詳細
